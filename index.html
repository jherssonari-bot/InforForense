Módulo 1: Fundamentos de Informática Forense

1. ¿Qué es la Informática Forense?

La informática forense (o cómputo forense) es una disciplina fascinante que combina la ciencia computacional con leyes y procedimientos legales. Su objetivo no es solo "encontrar datos", sino hacerlo de una manera que sea admisible ante un tribunal. Es la aplicación de técnicas científicas y analíticas especializadas a infraestructura tecnológica para identificar, preservar, analizar y presentar datos que sean válidos dentro de un proceso legal.

Objetivos Principales:

  • Recuperar evidencias perdidas, borradas o dañadas.
  • Rastrear la fuente de un ataque cibernético.
  • Prevenir futuros crímenes mediante el análisis de vulnerabilidades explotadas.

2. El Ciclo de Vida de una Investigación Forense

Este es el procedimiento estándar que debe seguirse rigurosamente para no invalidar la prueba.

A. Identificación

Determinar qué activos (laptops, servidores, móviles, nubes) contienen información relevante para el caso.

B. Preservación (Crítico)

Aislar y proteger la evidencia. Regla de Oro: Nunca trabajar sobre la evidencia original. Uso de bloqueadores de escritura.

C. Adquisición

Creación de una imagen forense bit a bit (copia exacta, incluyendo espacio "vacío" o borrado).

D. Análisis

El investigador trabaja sobre la copia usando herramientas para buscar palabras clave, recuperar archivos, etc.

E. Presentación / Reporte

Crear un informe técnico comprensible para abogados y jueces explicando qué se encontró.

3. Conceptos Clave

  • Cadena de Custodia: Documento legal que registra cronológicamente quién tuvo contacto con la evidencia.
  • Hashing (Huella Digital): Algoritmos (MD5, SHA-256) que generan un código único. Si cambia un bit, cambia el hash.
  • Principio de Locard: "Todo contacto deja un rastro".

4. Áreas de Especialización

  • Forense de Discos: Extracción de HDDs, SSDs, USBs.
  • Forense de Redes: Monitoreo de tráfico LAN/WAN.
  • Forense Móvil: Smartphones (WhatsApp, GPS).
  • Forense en la Nube: AWS, Azure, Google Drive.
  • Análisis de Malware: Ingeniería inversa de virus.

  • Volatility (RAM)
  • Wireshark (Redes)
  • Kali Linux (Modo Forense)

6. Retos y Carrera Profesional

Retos Actuales

Cifrado (Encryption): BitLocker, VeraCrypt.
Anti-Forense: Borrado seguro, esteganografía.
SSD y TRIM: El borrado físico rápido en discos modernos.

¿Cómo empezar? Aprende Sistemas Operativos a bajo nivel (Kernel, NTFS). Busca certificaciones como CHFI, GCFA o ACE. Practica con retos CTF forenses usando Autopsy.
